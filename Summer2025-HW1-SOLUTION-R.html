Sales and Customer Interaction Dataset

In this homework, you have been given a dataset that captures comprehensive details of 1,000 sales transactions made in 2024, focusing on various aspects of customer interactions and sales dynamics. It includes a mix of numeric and categorical variables, allowing for in-depth analysis.

Key features of the dataset include:

  • Customer_ID: Unique identifier for each customer.

  • Product_Category: Category of the purchased product (e.g., Electronics, Clothing, Home Appliances, etc.).

  • Purchase_Amount: Total amount spent on each purchase, in dollars.

  • Purchase_Date: Date of the transaction

  • Customer_Age: Age of the customer at the time of purchase.

  • Customer_Gender: Gender of the customer, Male or Female

  • Store_Location: Location of the store where the purchase occurred.

Response to Q5c

i) A confidence interval provides a range of plausible values for the mean response at a given value of the predictor. A prediction interval provides a range for a new individual response at that same predictor value, accounting for both uncertainty in the mean estimate and the variability of individual outcomes.

ii) The prediction interval is wider because it incorporates both the uncertainty in estimating the mean response and the additional variation due to individual error. In contrast, the confidence interval only reflects uncertainty in the mean.

iii) The 95% confidence interval of (10.2, 11.4) means we are 95% confident that the average sales for all observations with an advertising budget of 100 fall within this range. The 95% prediction interval of (7.5, 14.1) means we are 95% confident that a new observation with an advertising budget of 100 will have a sales value within that range.

iv) If the residual plot suggests non-constant variance (heteroscedasticity), then the models assumption of equal variance across levels of the predictor is violated. This can make prediction intervals unreliable because the estimated standard error varies with the predictor value. Applying a log transformation to the response variable can stabilize the variance, improving the accuracy and interpretability of interval estimates.

Question 7: Research Question (2 points)

(2 points) Research and explain the arguments that go into the predict function in detail. Please include object, newdata, interval, type, se.fit, and level.

Response to Q7

The predict function in R is used to make predictions based on a model object. Here are the key arguments that go into the predict function:

object: This is the model object you have created, such as a linear regression model (lm), generalized linear model (glm), or any other model object. It contains the fitted model that will be used for making predictions.
newdata: This is a data frame containing the new data for which you want to make predictions. The structure of newdata should match the structure of the data used to fit the model.
interval: This argument specifies the type of interval calculation. Common options include none (default), confidence, and prediction. These intervals provide additional information about the uncertainty of the predictions.
type: This argument specifies the type of prediction to make. For example, in the context of a generalized linear model, you might use response to get predictions on the scale of the response variable or link to get predictions on the scale of the linear predictors.
se.fit: This logical argument indicates whether standard errors of the predictions should be returned. If set to TRUE, the function will return standard errors along with the predictions.
level: This argument specifies the confidence level for the interval. The default is usually 0.95, corresponding to a 95% confidence interval.

Response to Q8 To determine and change the baseline (reference level) of a factor variable in R, you can use the levels and relevel functions. Heres how you can do it: Determine the Baseline First, you can check the levels of the factor to see what the current baseline is:

# Convert the variable to a factor

  3. 'Electronics'
  4. 'Furniture'
  5. 'Home Appliances'
  6. 'Toys'

The first level in the output is the current baseline. Change the Baseline To change the baseline, you can use the relevel function. For example, if you want to set Clothing as the baseline:

In[63]:
# Change the baseline to "Clothing"
Sales_dataset$Product_Category <- relevel(Sales_dataset$Product_Category, ref = "Clothing")

# Verify the change
levels(Sales_dataset$Product_Category)
  1. 'Clothing'
  2. 'Books'
  3. 'Electronics'
  4. 'Furniture'
  5. 'Home Appliances'
  6. 'Toys'

The specified category (Clothing) will become the new baseline.
